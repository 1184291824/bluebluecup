#					抢答器使用说明
##	1 【功能介绍】：
1. 本抢答器具有多种功能
2. 模式1：基本倒计时抢答模式，主持人按下开始按钮后，选手开始抢答
3. 模式2：答题时间倒计时模式，主持人按下开始按钮后，选手开始作答
4. 模式3：查看选手得分
5. 模式4：设置抢答时间和答题时间


##	2 【按键说明】：
		
### [S5]:切换模式按钮

### {模式 1}：
- [S2]:开启定时器
- [S3]:暂停定时器

### {模式 2}：
- [S2]:开启定时器
- [S3]:回答错误
- [S4]:回答正确（注意：按一次加分一次，请勿多次按下）

### {模式 3}：
- 无效

### {模式 4}：
- [S2]：增加时间
- [S3]: 减少时间
- [S4]: 切换所设置的时间
